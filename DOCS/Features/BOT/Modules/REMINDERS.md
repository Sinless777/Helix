# **15. Reminders**

The Reminders module enhances the functionality of your Discord server by allowing users to set reminders. With this module, users can create custom reminders for important events, tasks, or any other time-sensitive information. Here's an overview of the Reminders module's features and the required information to utilize it effectively:

## Features

The Reminders module typically includes the following features:

1. **Reminder Creation**: Users can create reminders with customizable content. They can specify the details of the reminder, such as the event, task, or any other information they want to be reminded about.

2. **Timing and Scheduling**: Users can set the timing for their reminders. They can specify the date, time, and even recurring schedules if needed. This ensures that they receive timely reminders for their important events or tasks.

3. **Delivery Options**: The module provides options for delivering reminders. Users can choose to receive reminders through direct messages or have them posted in designated channels, allowing for flexibility and convenience.

## Required Information

To set up and utilize the Reminders module, the following information is typically required:

1. **Reminder Content**: Specify the content of the reminder. This can be any text or information that the user wants to be reminded about. For example, it could be a description of an event, a task to complete, or any other relevant information.

2. **Timing or Schedule**: Determine the timing or schedule for the reminder. This can be specified using a specific date and time (datetime) or a duration from the current time. Users can set a one-time reminder or recurring reminders based on their requirements.

## Optional Requirements

Depending on the capabilities of the Reminders module, you may have additional optional requirements or preferences, such as:

- **Delivery Method**: Specify whether you want the reminders to be delivered through direct messages or posted in designated channels. This allows users to choose their preferred method of receiving reminders.

Please note that the specific implementation and configuration details of the Reminders module can vary depending on the Discord bot you are using. It is recommended to consult the official documentation or support channels of the bot for accurate instructions on how to enable and configure the Reminders module.

If you have any further questions or need assistance, feel free to reach out to the bot's support channels or refer to the relevant module documentation.
