# Auto Purge (PREMIUM)

The Auto Purge module is a premium feature that enables automatic purging of messages in Discord channels. It provides a convenient way to automatically delete messages based on specified criteria, such as their age or content. The module offers customization options and rules to fine-tune the purging behavior.

## Features

The Auto Purge module includes the following features:

- **Automatic Message Deletion**: Messages in Discord channels can be automatically deleted based on specified criteria. This can include deleting messages that are older than a certain duration, contain specific keywords, or meet other defined conditions.

- **Customizable Purging Settings**: The module allows customization of the purging settings to fit the server's needs. This can include specifying the duration after which messages should be deleted, defining specific keywords to look for in messages, or other parameters depending on the capabilities of the bot.

- **Optional Advanced Filtering**: The Auto Purge module may provide advanced filtering options to further refine the purging behavior. This can include exemptions for specific users or roles, custom filters based on regular expressions, or other advanced criteria to selectively delete messages.

- **Premium Feature**: Some Discord bots may offer additional advanced filtering options and exemptions as part of their premium offering. These premium features provide enhanced control and customization over the purging process.

## Required Information

To configure the Auto Purge module, you need to provide the following information:

- **Purging Criteria or Rules**: Specify the criteria or rules that determine which messages should be deleted. This can be provided as an object or JSON, depending on the configuration options of the bot. Define the criteria based on message age, content, or any other relevant factors.

- **Optional: Advanced Filtering Options or Exemptions**: If available as a premium feature, the Auto Purge module may offer advanced filtering options or exemptions. These options can be provided as an object or JSON and allow for more granular control over the purging process. Examples of advanced filtering options include exemptions for specific users or roles or the use of custom filters based on regular expressions.

The implementation and configuration details of the Auto Purge module may vary depending on the Discord bot you are using. It is recommended to consult the official documentation or support channels of the bot to get accurate instructions on how to set up and configure the Auto Purge module.
