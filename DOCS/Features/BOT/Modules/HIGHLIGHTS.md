**25. Highlights**

The Highlights module provides a functionality that allows users to highlight specific messages or keywords within a Discord server. This feature helps draw attention to important information or specific content within a channel. Here's an overview of the Highlights module's features and the required information to set it up:

## Features

The Highlights module typically includes the following features:

1. **Marking Messages or Keywords as Highlights**: Users can mark certain messages or keywords as highlights, indicating their significance or relevance. This can be done through a specific command or by interacting with the message itself.

2. **Notification for Highlighted Messages or Keywords**: When a highlighted message or keyword is detected within the server, the module can generate notifications to alert relevant users or designated channels. This ensures that important information doesn't go unnoticed.

3. **Optional Customization Options and Notifications**: Depending on the capabilities of the bot or module, there may be customization options available. These options can include choosing the type of notifications (e.g., mentions, direct messages, channel alerts), defining highlight colors or styles, or specifying which users or channels receive the notifications.

## Required Information

To configure and utilize the Highlights module, the following information is typically required:

1. **Messages or Keywords to be Highlighted**: Specify the messages or keywords that should be marked as highlights. This can be a single keyword, a list of keywords, or specific message IDs. Highlighting messages can be done manually or through commands provided by the bot.

2. **Optional: Notification Settings or Customization Options**: If the Highlights module offers customization options or notification settings, provide any preferences or requirements you have. This can include specifying the notification recipients, choosing the notification type, customizing highlight colors or styles, or defining the format of the notification messages.

Please note that the specific implementation and configuration details of the Highlights module can vary depending on the Discord bot you are using. It is recommended to consult the official documentation or support channels of the bot for accurate instructions on how to enable and configure the Highlights module.

For further assistance or questions, please refer to the respective module documentation or reach out to the bot's support channels.
