**16. StarBoard**

The StarBoard module enhances your Discord server by allowing users to star or highlight messages that they find interesting or important. These starred messages are then automatically forwarded to a designated StarBoard channel, creating a curated collection of noteworthy content. Here's an overview of the StarBoard module's features and the required information to set it up:

## Features

The StarBoard module typically includes the following features:

1. **Starring or Highlighting Messages**: Users can star or highlight messages they consider noteworthy or want to share with the community. This can be done by reacting to the message with a specific emoji or by using a designated command.

2. **Automatic Forwarding**: When a message reaches a certain number of stars or highlights, it is automatically forwarded to the StarBoard channel. This ensures that the most popular or significant messages are prominently displayed in a centralized location.

3. **Optional Configuration Settings**: The StarBoard module may provide optional configuration settings. One common setting is the star count threshold, which determines the minimum number of stars or highlights required for a message to be forwarded to the StarBoard channel. This threshold can be customized based on your server's preferences.

## Required Information

To set up and utilize the StarBoard module, the following information is typically required:

1. **StarBoard Channel**: Specify the channel where the starred messages will be forwarded. Provide the channel ID or name to designate the StarBoard channel.

2. **Reaction or Command**: Determine the specific reaction or command that users should use to star or highlight messages. This can be an emoji reaction (such as ⭐) or a custom command (e.g., "!star"). Inform your users about the designated method for starring messages.

3. **Optional: Star Count Threshold**: Decide on the star count threshold that triggers the forwarding of messages to the StarBoard channel. This can be a specific number (e.g., 5 stars) or a percentage of the total number of server members. If you don't specify a threshold, all starred messages may be automatically forwarded.

Please note that the specific implementation and configuration details of the StarBoard module can vary depending on the Discord bot you are using. It is recommended to consult the official documentation or support channels of the bot for accurate instructions on how to enable and configure the StarBoard module.

For further assistance or questions, please refer to the respective module documentation or reach out to the bot's support channels.
