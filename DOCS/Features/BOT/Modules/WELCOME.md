**13. Welcome**

The Welcome module provides functionality to send welcome messages to new members who join a Discord server. It helps create a warm and inviting atmosphere for newcomers. Here's an overview of the Welcome module's features and the required information to set it up:

## Features

The Welcome module typically includes the following features:

1. **Customizable Welcome Messages**: You can customize the content of the welcome messages that will be sent to new members. This allows you to personalize the message and make it more engaging for newcomers.

2. **Personalization Options**: The Welcome module often provides personalization options based on user information. You can include variables like the username, join date, or any other relevant details in the welcome message to make it feel more personalized to each user.

3. **Optional Role Assignment**: Additionally, the Welcome module may offer the ability to assign roles to new members upon joining. This can be useful for granting specific permissions or access to channels immediately after joining the server.

## Required Information

To set up the Welcome module, you will need the following information:

1. **Welcome Message Content**: Specify the content of the welcome message that will be sent to new members. This can be a string of text or a predefined message template that supports variables for personalization.

2. **Discord Channel for Sending Welcome Messages**: Identify the Discord channel where the welcome messages should be sent. Provide the channel name, ID, or snowflake value to ensure the messages are delivered to the correct location.

3. **Optional: Role ID or Name for Role Assignment**: If you want to assign a role to new members upon joining, provide either the role ID or name that should be assigned. This ensures that new members receive the appropriate role and associated permissions.

Please note that the implementation and configuration details of the Welcome module can vary depending on the Discord bot you are using. It is recommended to consult the official documentation or support channels of the bot for accurate instructions on how to enable and configure the Welcome module.

For further assistance or questions, please refer to the respective module documentation or reach out to the bot's support channels.
