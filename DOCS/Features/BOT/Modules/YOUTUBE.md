**YouTube (PREMIUM)**

The YouTube module offers functionality that allows users to subscribe to a YouTube channel and receive updates in a designated Discord channel. This integration enhances the communication between YouTube content creators and their Discord community. Here are the key features and the required information to set up the YouTube module:

## Features

The YouTube module typically includes the following features:

1. **YouTube Channel Subscription**: Users can subscribe to a specific YouTube channel within Discord. This ensures that they receive updates about new videos, live streams, or other content posted on the channel.

2. **Posting Updates to Discord**: When a new video or content is published on the subscribed YouTube channel, the YouTube module automatically posts an update to a specified Discord channel. This enables the Discord community to stay up-to-date with the latest content from their favorite creators.

3. **Message Format Options**: The YouTube module offers flexibility in how the updates are presented in the Discord channel. It can support both plain text messages or more visually appealing embedded messages. This allows for customization and alignment with the server's style and preferences.

## Permissions

To utilize the YouTube module, users generally require the "Manage Server" permission. This permission ensures that users have the necessary access to set up and configure the module within the Discord server.

## Required Information

To set up the YouTube module, you will need the following information:

1. **YouTube Channel Information**: Provide the necessary details about the YouTube channel, such as the channel ID or channel name, to specify which channel the module should subscribe to.

2. **Discord Channel for Posting Updates**: Identify the Discord channel where the YouTube updates should be posted. Provide the channel name, ID, or snowflake value to ensure the updates are delivered to the correct location.

3. **Options for Message Format**: Decide whether the updates should be posted as plain text messages or embedded messages. If you choose embedded messages, you may need to provide additional configuration details or formatting options.

Please note that the YouTube module is often available as a premium feature in Discord bots. This means that some of its functionalities may require a subscription or payment. It is recommended to consult the official documentation or support channels of the bot you are using for accurate instructions on enabling and configuring the YouTube module.

For further assistance or questions, please refer to the respective module documentation or reach out to the bot's support channels.
